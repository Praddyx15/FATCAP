# Tests

## Test 1 - Homepage Loading Smoke Test [sanity]
Viewport: desktop (1280x800)
Description: Verifies that the FATCAP studio landing page loads successfully, displays the main logo or custom background canvas, and renders the primary navigation links.
Steps:
1. Navigate to "/"
2. Wait 2 seconds for animations to initialize
3. Check that the page title contains "FATCAP"
4. Verify the navigation bar with link "WORK" is visible
Expected:
The page title should be "FATCAP — Motion & Design Studio" and the navigation link "WORK" should be present on the page.

## Test 2 - Work Section Page Navigation
